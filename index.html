Video Game Projects

Dreams of Royalty

GAME DESIGNER | GAME PROGRAMMER

Dreams of Royalty is a medieval narrative experience build into a video game developed in Unity and using C#.
The game takes place in a medieval castle, where our main character, Channah, has just arrived to work. Channah always dreamed about being a king, and now that he is in the castle, he'll do what is needed to acomplish that goal.
Investigate, explore, talk and spy to take advantage of different situations.

This current version of the game is a prototype.
